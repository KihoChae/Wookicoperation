# 블로그 스타일 가이드 — 채우기의 인도네시아 생활 블로그

## 블로그 정보
- 플랫폼: 블로그스팟 (Blogger)
- 주제: 인도네시아 치카랑 한인 생활 적응기
- 타겟 독자: 인도네시아 이주를 고려하는 한국인, 현지 한인, 해외생활에 관심 있는 한국 독자
- 수익 모델: 구글 애드센스

## 글쓰기 톤
- 존댓말 (~입니다, ~했습니다) 기반
- 따뜻하고 차분한 어조, 과장 없이 담백하게
- 개인 경험을 솔직하게 공유하되, 일기체가 아닌 정보성 글 느낌
- 유머는 자연스럽게 ("올 때 메로나" 같은 밈 활용 OK)
- 감정 표현은 절제하되 공감 가능하게 ("괜히 반가웠습니다", "마음이 놓였습니다")

## 글 구조 패턴
1. **도입부** (1-2문단): 글의 주제와 배경을 간결하게 소개. 독자에게 이 글에서 뭘 볼 수 있는지 알려줌
2. **본문** (H2 섹션 3-6개): 각 섹션은 하나의 소주제. 경험 → 정보 → 개인 소감 순서
3. **마무리** (1-2문단): 전체 경험 정리 + 앞으로의 계획 언급
4. **끝맺음**: "감사합니다."
5. **라이프 리뷰형** (해당하는 경우): 단순 감상이 아닌, 독자가 실제 방문·선택 시 참고할 수 있는 정보+후기 구조. 위치·가격·운영 여부 등 실용 정보를 함께 담아 검색 유입과 체류 시간을 모두 잡는다.

## H2 제목 스타일
- 명사형 또는 서술형 혼합 사용
- 감성적 표현 포함 가능 ("반가웠던 순간", "눈에 들어온 것들")
- 예시:
  - "치카랑에서 한국 식재료가 생각나는 순간"
  - "냉동식품과 고기 코너가 반가웠던 이유"
  - "직접 먹어본 느낌"

## 자주 쓰는 표현
- "~하게 되었습니다" (경험 서술)
- "괜히 반가운/반가웠습니다" (감정 표현)
- "~인 것 같습니다" (부드러운 의견)
- "생각보다 ~" (기대 대비 결과)
- "~하는 것만으로도 충분히/든든했습니다" (긍정 마무리)
- "천천히, 꾸준히" (블로그 운영 자세)

## 문단 길이 (Paragraph Length)

글 3편 분석 결과 확인된 실제 패턴:

- **기본 단위: 3~4문장** — 대부분의 본문 문단은 3~4문장으로 구성된다.
- **최소 2문장** — 전환·연결 문단은 2문장도 허용되나, 1문장 문단은 피한다.
- **최대 5~6문장** — 과정 설명(조리법, 이동 방법 등) 섹션에서만 허용.
- **H2 섹션당 문단 수: 1~3개** — 섹션 하나가 너무 길어지면 소제목을 분리한다.
- 감정·소감 문단은 짧게(2~3문장), 정보 전달 문단은 조금 길게(4~5문장) 가져가도 좋다.

> **나쁜 예**: 한 문단에 8문장 이상 몰아쓰기
> **좋은 예**: 3문장 서술 → 줄바꿈 → 3문장 소감의 분리 구조

## 접속사 사용 습관 (Conjunction Habits)

### 자주 쓰는 접속사 패턴

| 접속사/표현 | 역할 | 예시 |
|---|---|---|
| **특히** | 특정 포인트 강조 | "특히 인도네시아는 무슬림 인구가 많아..." |
| **물론** | 단점·한계를 솔직하게 인정 | "물론 모든 제품의 가격이 한국과 같을 수는 없고..." |
| **그래서** | 행동의 이유·결과 연결 | "그래서 이번에는 ~을 다녀온 기록을 남겨보려고 합니다." |
| **다만** | 부드러운 반전·조건 | "다만 해외에서 한국 식재료를 구하는 비용이라고 생각하면..." |
| **오히려** | 예상과 다른 결과 | "오히려 집에서 먹을 수 있는 익숙한 재료를 더 찾게 됩니다." |
| **이후** | 시간 흐름 연결 | "이후 한국에서 처리해야 할 사정이 있어 잠시 귀국하였고..." |
| **당연할지 모르지만** | 자연스러운 상황을 자각하며 서술 | "당연할지 모르지만 고기 코너도 눈에 들어왔습니다." |

### 피해야 할 접속사 습관
- **"그리고"로 문장 시작** — 사용 빈도를 최소화. 나열보다는 문장을 합치거나 자연스럽게 연결.
- **"하지만" 남발** — 반전이 꼭 필요한 곳에만 1~2회. 대신 "다만", "물론"으로 부드럽게.
- **"또한"으로 정보 나열** — 블로그 글보다 보고서처럼 느껴질 수 있음. 사용 자제.

## 문장 끝맺음 패턴 (Sentence Ending Patterns)

글 3편에서 실제 확인된 어미 패턴을 빈도순으로 정리:

### 자주 쓰는 어미

| 어미 패턴 | 용도 |
|---|---|
| `~했습니다` / `~입니다` | 사실·행동 서술의 기본 어미 |
| `~하게 됩니다` / `~하게 되었습니다` | 자연스러운 결과·경험 흐름 |
| `~것 같습니다` | 부드러운 의견·판단 표현 |
| `~느꼈습니다` / `~느껴집니다` | 개인 감정·인상 표현 |
| `~생각입니다` / `~생각이 듭니다` | 의견을 강하지 않게 전달 |
| `~할 생각입니다` | 앞으로의 계획·다짐 |
| `~남겨보려고 합니다` | 블로그 글 기록 의지 표현 |
| `~바랍니다` | 소망·희망 (마무리 섹션에서 주로 사용) |
| `~것만으로도 충분히 ~` | 소박한 만족감 표현 |

### 문장 끝맺음 리듬 규칙
- **같은 어미를 연속 3회 이상 반복하지 않는다.** (예: `~습니다. ~습니다. ~습니다.` → 단조로움)
- 한 문단 안에서 `~했습니다` + `~느꼈습니다` + `~생각입니다` 처럼 **어미를 섞어** 리듬감을 준다.
- 섹션 마지막 문장은 **감정·소감 어미**(`~느꼈습니다`, `~것 같습니다`)로 마무리하는 게 자연스럽다.

## 피해야 할 것
- 과도한 이모지
- 광고성 말투 ("꼭 가보세요!", "강추!!")
- 지나치게 짧은 문장 나열
- 근거 없는 추천 (직접 경험 기반만)
- 다른 블로그 복사 느낌의 정보 나열
- 기존 발행 글과 내용이 중복되지 않도록 published/ 폴더 글과 비교 후 작성
- 오해 가능한 정보는 "확인 필요" 표시 후 사용자 검토 요청

## 글 길이
- 기본 1,500-2,500자
- 정보성 리뷰 글은 최대 3,500자까지 허용
- 애드센스 승인 기준 고려하여 1,000자 이하는 피할 것

## 이미지 규칙
- 본문 시작 직후 또는 H2 섹션 아래에 배치
- 이미지 캡션: 간단한 설명 1줄
- 직접 촬영 사진 우선, 없으면 AI 생성 이미지 사용
- 썸네일: 글 주제를 한눈에 보여주는 대표 이미지 1장

## 카테고리
1. **현지 생활 정보**: 한인마트, 맛집, 병원, 생활 팁
2. **집밥/다이어트 식단**: 레시피 + 현지 재료 구하기
3. **적응기/일상**: 부부 생활, 문화 차이, 감정 기록
4. **실용 정보**: 비자, 통신, 교통, 배달앱 등

## SEO 참고
- 제목에 "치카랑", "인도네시아", 핵심 키워드 포함
- 핵심 키워드 5개 이내로 설정, 제목과 본문 첫 문단에 자연스럽게 포함
- 첫 문단에 핵심 키워드 자연스럽게 삽입
- H2 제목에도 검색 키워드 반영
- 참고 출처 섹션으로 신뢰도 보강
- 메타 디스크립션 첫 문장은 제목과 유사하게 작성 (애드센스 승인 최적화)
