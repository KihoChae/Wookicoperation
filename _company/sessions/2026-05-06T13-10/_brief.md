COSLAB Hub는 화장품 ODM 사업 전반을 컨트롤하고 기록 관리하는 통합 SaaS 플랫폼이야.
단순히 BPOM/할랄 인허가 관리만 하는 게 아니라, ODM 사업의 모든 프로세스를 하나의 시스템에서 관리해.

【관리 범위 — ODM 사업 전 주기】

원부자재 관리 (Materials/INCI)
- 원료 마스터 데이터, INCI명, 공급처, 가격 이력 관리

포뮬러/레시피 관리 (Formula Management)
- base_formulas 기반 처방 개발, 버전 관리, 상태 추적
- 자동 진척 매핑 (완료 처방 수 자동 카운트)

안전성 평가 (Safety Assessment)
- 원료별 안전성 평가 데이터 관리

안정성 시험 (Stability Testing)
- 시험 일정, 측정 데이터, 결과 기록

프로덕트 BOM 관리 (Product/BOM)
- 제품별 BOM 구성, 원가(COGS) 이력 관리, 출시 상태 추적

영업 관리 (Sales Pipeline + 견적)
- 고객 파이프라인 L1~L4 칸반 관리
- 견적 산출, 계약 관리, 매출 인식 (Invoice 발행 시점)

주문 및 생산 프로세스 관리
- PO 단위 + SKU 단위 리드타임 추적
- 개발 리드타임 (First Proposal → 제품 확정)
- 공급 리드타임 (PO 접수 → 최종 납품)

BPOM/할랄 인증 문서 관리
- 인허가 프로세스 전체 추적
- 필수 문서 상태 관리, 제출 이력

라벨 검수 (Label Check)
- 제품 라벨 규정 준수 확인

문서 관리 (Documents)
- 전사 문서 통합 관리, SOP 완성도 추적
- 23종 필수 SOP 마스터 관리

직원 교육 현황 관리 (Talent + Training)
- 인재 정보, 외부/내부 교육 이수 현황
- 퇴직/휴직 시 자동 아카이브 + 업무 재배치 알림

프로젝트 매니지먼트 (PM Module — 최신 v4)
- OKR 기반 성과 관리: 회사 KPI → 개인 PIP → KR → Initiative → Task
- 일별 진척 스냅샷 (매일 02:00 자동)
- Stage-Gate 전환 (D-M-R-B-P-S)
- 체크리스트 시스템 (항목별 evidence 설정, 일일/주간/월간 반복)
- Biweekly Report AI 자동 초안 → 검수 → 발행
- 데일리 스크럼 AI 생성
- 분기 회고 + KR 둔화 자동 플래그
- 전직원 대시보드: /pm 첫 화면에서 회사 KPI 트리 + 본인 기여도 확인

히스토리 관리
- 모든 변경 사항 audit log 기록 (누가, 언제, 무엇을)
- 월별 파티셔닝으로 대용량 처리
