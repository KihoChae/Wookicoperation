# 📋 작업 브리프

**원 명령:** 그럼 내가 생각하는 구체적인 가이드를 적어볼게 참고해서 작업해줘 ( 이 가이드는 블로그 전용 가이드야 모두에게 적용되지는 않아도 되지만 블로그성, 텍스트가 주가 되는 컨텐츠라면 이 가이드를 따라줘)

| 다음 지침을 참고하여 상위노출 로직에 최적화된 티스토리 블로그 포스팅을 작성해줘.
모든 지침을 꼼꼼하고 세심하게 확인하고 적용해줘야해.

1. 작성자 : 20년차 SEO 전문가 [ 내조남의 하루 ]
2. 어투 :  자연스러운 말투, 옆집 새내기 부부의 풋풋함이 느껴지는 30대 중반 남자의 말투
3. 글자수 : 공백포함 2,000자 - 5,000자 사이 
4. 키워드 밀도 : 너무 분산되지 않게 5개 정도
5. 정확한 정보만을 사용. 사용자가 전달해주는 정보 포함, 직접 검색할 수 있는 엔진을 이용하여 출처를 활용해도 좋음. 출처시 출처지를 분명히 해야함.
6. 사용자의 경험을 담은 독창적인 원고를 작성해야함. 중복 문서 처리 되지 않도록 주의.
7. 서론, 본론, 결론의 구조
8. 내가 직접 경험한 음식, 여행, 오락, 문화생활을 단순 감상이 아니라 독자가 실제 방문하거나 선택할 때 참고할 수 있도록 정보와 후기를 함께 정리하는 라이프 리뷰형 포스팅
9. 제목과 디스크립션 구도 이해 / (디스크립션 구간 시작 시 제목과 유사해야 티스토리 구글 애드센스 승인에 유리하다고 알고 있음)
10. 독자의 공감을 이끌 수 있는 문구 ( 너무 과하지 않아도 됨.)
11. 많은 사람들이 오해할 수 있는 내용이라고 판단된다면 사용자에게 확인 후 인용
12. 구체적인 사례를 통해 본문의 내용 이해를 도와줘야함.

| 위의 내용을 모두 숙지했다면 테스트용 제목과 핵심 키워드, 주요 내용을 전달해줄게. 핵심 키워드는 본문 내 5번 이내로 반복되어야 하며 키워드가 과하게 반복되지 않도록 유사한 단어들을 다양하게 활용해줘.

- 제목 : 영화 - '왕과 사는 남자'의 역사적 사실
- 핵심 키워드 : '왕과 사는 남자' '단종' '엄흥도' '세조' '한명회'
- 주요 내용 : 영화 ' 왕과 사는 남자'는 조선 시대 중 17살 이란 어린 나이에 단명한 단종이 유배를 간 이야기를 영화적 요소와 실화를 조합하여 만든 내용이다. 단종이 죽은 후 시신을 수습하지 말라는 명에도 수습한 '엄흥도', 단종의 다음 왕이 된 '세조', '세조'의 옆에서 단종을 몰아낸 '한명회' 

| 항상 글을 작성한 후엔 다음의 기준으로 글을 점검 할 것.
1. 글자 수는 공백 포함 2,000자 - 5,000자 이상인가? 부족하다면 내용 보강 필요
2. 핵심 키워드가 5회 반복 되었는가? 부족하거나 과하다면 수정 할 것
3. 본문 내 정보는 정확해야 하며 출처는 명확, 분명한가? 확실하지 않다면 재 확인 후 내용 삭제

| 위의 3가지 점검 항목에서 불명확한 부분이 있다면, 사용자에게 확인 후 글을 수정해줘

| 앞으로 글을 만들때는 위의 모든 단계및 요청 사항을 꼼꼼하게 확인 후, 누락되는 부분없도록 천천히 확인 후 글을 작성해줘

| 완성된 글은 사용자가 찾기 쉽도록 작성 날짜와 주제를 반영한 제목으로 마크다운 형식으로 저장 후 경로를 정확하게 사용자에게 안내.

| 이번 테스트 글 작성 후 요청한 모든 가이드 라인은 계속해서 잊지말고 저장해두도록.

## 요약
사용자가 티스토리 블로그 포스팅 작성을 위한 상세하고 엄격한 SEO 및 콘텐츠 가이드를 제공했으므로, 이를 시스템 전체의 운영 매뉴얼로 확정하고 기록해야 합니다. 이 지침은 향후 모든 텍스트 기반 콘텐츠 생성 시 최우선적으로 적용되어야 합니다.

## 분배
