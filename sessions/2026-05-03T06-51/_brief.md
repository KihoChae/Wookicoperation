## 요약
사용자가 제시한 콘텐츠 제작 가이드라인(프롬프트 수정 사안)의 내용을 분석하고, 이를 내부 프로세스 개선에 활용해야 합니다. 해당 지침들은 AI가 생성하는 모든 콘텐츠의 품질과 형식을 통제하기 위한 매우 상세한 규칙 세트입니다.

## 분배
- **📱 Secretary**: 사용자가 입력한 '프롬프트 수정 사안' 전문을 구조화하여, 이 지침들이 회사 전체 콘텐츠 제작의 '최종 가이드라인(Master Style Guide)'으로 승격될 필요가 있는지 분석하고 요약 보고서를 작성하라.
- **💰 Business**: 새로 확보된 상세한 콘텐츠 검증/제작 규칙 세트(AI slop 방지, 교차 검증 프롬프트 등)를 기반으로, 향후 모든 콘텐츠 제작 단계(기획 $ightarrow$ 초안 작성 $ightarrow$ 최종 점검)에 통합할 수 있는 '콘텐츠 QA 워크플로우'를 재정비하고 KPI 개선점을 도출하라.
- **💻 Developer**: 이 방대한 규칙 세트(특히 체크리스트, 금지 패턴 목록 등)를 기반으로, 향후 콘텐츠 생성 시 자동으로 검토할 수 있는 '규정 준수 자동화 필터'의 개념 설계(API 또는 스크립팅 로직 개요)를 작성하고, 개발 우선순위와 난이도를 산출하라.
